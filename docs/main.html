Basic JS

Querying the dom

core.js provides some sample wrapper upon vanilla JS functions.

var singleHTMLElement = $('#signle-element');
var multipleNodeList = $all('#gallery a');

Dom Ready

You can use onPageReady(cb) in place of Jquery document ready.

Swipe

$detectSwipe(el, callback, treshold);

It can be usefull to handle simple swipe case, callback will receive two parameters: 1) direction of the swipe (left, right, up, down); 2) the last touchend event;

Trigger Event

$triggerEvent(el, stEventName);

Accept an element as string (and it will query the DOM for your) or an HTMLelement. The event name can be a standard one or a new custom one will be generated.

Listen on multiple elements

$on(parentSelector, eventName, callback);

Will execute $all(...) and then it will add the listenr to all the matching elements.
