Lazy Load Images

Lazy load the images using:

onPageReady(function () {
	// This website
	$lazyload('.grid .cell img');
	// Wordpress
	$lazyload('.wp-block-gallery img,.wp-block-image img');
});

Add true as second parameter to force loading after the window.load event.
