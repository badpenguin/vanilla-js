How it Works

In single mode you can just watch a single photo, any mouse or keyboard event will close the modal

In gallery mode you can just watch multiple photo, starting from the one you clicked.

You can navigate with keyboard (arrow left, arrow right, space or escape) or swipe or clicking the image.

Going past the last photo or before the first one then it will close the modal.

Swiping up or down will close too.

Keyboard escape also closes the modal.

Using with Wordpress

To have one lightbox for all the clickable gallery and images of the page:

$lightbox('.wp-block-gallery,.wp-block-image', 'a');

To have a single lightbox for each one, please assign an "id", or a unique class to every element:

$all('.wp-block-image,.wp-block-gallery').forEach(function (v, i) {
	var name = 'lightbox-instance-' + i;
	v.id = name;
	el = $lightbox('#' + name, 'a');
});

Using with other engines

The script detected the URL to open as modal looking the the attributes: data-src, href, src

Captions are detected looking at the attributes: data-title, title, alt

This way, adding data-src, you can use it with single images without wrapping with an anchor:

$lightbox('IMG.lightbox');
